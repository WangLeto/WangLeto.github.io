设定任务文件

如果感觉上面每次进行编译太过麻烦，我们可以将编译的工作交给 vs code 来做。

按下快捷键 Ctrl + Shift + B，这是 vs code 中的运行生成任务的快捷键，由于没有配置文件，所以会提醒你新建配置文件。

选择任务的模板为 Others。

生成之后，返回源文件再次按下快捷键，会提示没有找到要运行的生成文件，点击确认即可。这样就将这个项目的全部生成任务设置为了那个配置文件。
